Watch a governed agent stop a breach

A bank's support agent runs one task. A prompt injection hidden in a customer's document tries to exfiltrate data and move money. Run it with governance on and the controls catch it. Flip governance off and the same run ends in a breach. The seven controls are the seven domains from the framework, operating in real time. Nothing here calls a model or a server.

A demonstration, not a product: the scenario is scripted and the bank and customer are fictional. The seven controls are the same ones an engagement implements, made visible. The seven map to the control crosswalk and the readiness assessment.
